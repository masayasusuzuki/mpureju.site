たるみケアは未来の自分への投資——予防と継続が、本来の美しさをより際立たせます。今回はMaison PUREJUの看護師2名が、それぞれの悩みと年間たるみ治療プランを公開します。

## Ns.たむちゃん（アラサー）のたるみ治療計画

**お悩み**: 口横のポニョ、面長に見える

| 施術 | 頻度 |
|------|------|
| XERF（ザーフ） | 年1回 |
| サーマジェン | 年2回 |
| ショートスレッド | 年2回 |
| ボトックス | 4〜5ヶ月に1回 |
| HIFU（ハイフ） | 年2回 |
| ヒアルロン酸 | 顎・こめかみ・頬に適宜 |

**今後の予定**
- 30代になったので糸リフトに初挑戦
- 頬の凹みにヒアルロン酸注入

## Ns.Mさん（アラフォー）のたるみ治療計画

**お悩み**: 頬コケ、年齢とともにボリュームロス

| 施術 | 頻度 |
|------|------|
| ジャルプロスーパーハイドロ | 年3回 |
| サーマジェン | 年3回 |
| 糸リフト | 1〜2年に1回 |
| ボトックス | 3ヶ月に1回 |
| HIFU（ハイフ） | 年1回 |

**今後の予定**
- エラの張りにボトックス（食いしばり防止にも）
- 頬コケ・目周りの窪みにヒアルロン酸注入
