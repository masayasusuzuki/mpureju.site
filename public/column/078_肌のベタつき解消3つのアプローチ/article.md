夏になると気になる肌のベタつき。皮脂が過剰に分泌されると、テカリ・ニキビ・毛穴の開きの原因になります。夏に多い原因として、紫外線やエアコン乾燥によるバリア機能の低下があります。バリア機能が低下すると肌の水分が失われ、それを補うために皮脂が過剰分泌されます。

## ベタつきを抑える3つのアプローチ

### 💊 内服

| 成分 | 働き |
|------|------|
| ビタミンBサプリ | 皮脂の分泌を抑え、皮膚を強化 |
| 亜鉛サプリ | 皮脂の過剰分泌を抑え、ターンオーバーを促進 |
| ビタミンCサプリ | 皮脂の分泌と酸化を抑え、毛穴を目立ちにくくする |

### 🧴 外用（ゼオスキン）

| 製品 | 働き |
|------|------|
| バランサートナー | 余分な皮脂・角質を取り除き肌を滑らかに整える |
| Cセラム | ビタミンC配合で皮脂を抑制 |
| スキンブライセラム | ターンオーバーを促進し皮脂分泌を正常化 |
| ロザトロール | 過剰な皮脂と赤みを抑える |

### 🤖 施術

| 施術 | 働き |
|------|------|
| ポテンツァ | 皮脂腺に働きかけ皮脂分泌をコントロール |
| うるおいシルキー注射 | 真皮を強化し汗や皮脂を抑制 |
| ボトックス | 皮脂腺・汗腺に作用して汗と皮脂を抑制 |
| ピーリング | 余分な皮脂・角質を除去しターンオーバーを促進 |
