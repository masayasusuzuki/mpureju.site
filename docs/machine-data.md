# microCMS 医療機器データ（コピペ用）

microCMS `machines` API に投入するデータ一覧。
投入完了後このファイルは削除する。

---

## API設定

- API名: `医療機器`
- エンドポイント: `machines`
- 型: リスト形式

---

## スキーマ

| フィールド | 種別 | 説明 |
|---|---|---|
| name | テキスト | マシン名（日本語） |
| name_en | テキスト | マシン名（英語） |
| slug | テキスト | URLスラッグ |
| thumbnail | 画像 | サムネイル（16:9） |
| type | テキスト | 技術種別（HIFU / RF / ニードルRF / レーザー / CO2レーザー / 光治療 / LEDライト / エレクトロポレーション） |
| catch_copy | テキスト | キャッチコピー |
| target_concerns | テキスト | 対象のお悩み（たるみ / 毛穴・ニキビ跡 等） |
| description | リッチエディタ | 詳細説明 |
| sort_order | 数値 | 表示順 |
